
< 스마트팜 프로젝트 >
- 프로젝트명(가칭): AnyGrow2 스마트팜 통합 제어 시스템
- 목적: 실내 재배 환경(온도/습도/CO₂/조도)을 실시간 모니터링하고, 조명 스케줄/장치 제어(LED·양액펌프·UV)를 통해 생육 조건을 안정적으로 유지하는 데스크톱 기반 스마트팜 솔루션.
- 센서(온도·습도·CO₂·조도) 데이터를 시리얼로 받아 실시간 수치/그래프로 모니터링하고 이상 구간을 색으로 표시한다.
- 조명은 작물별 프리셋 + 수동 스케줄(자정 넘어가는 구간 포함)로 설정하며, 주기적으로 현재 시간에 맞는 모드를 계산해 자동 적용한다.
- LED(Off/Mood/On)는 제어 가능하고, 시리얼 재연결로 통신 장애를 대응하며 펌프/UV 같은 추가 장치는 UI는 있으나 패킷은 미확정 상태다.
